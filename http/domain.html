# 域名

又称网域，根据区域划分的网络；

所有域名均是有ICANN管理的。前段时间ICANN整了个顶级域名开放申请，宣称各机构均可以申请顶级域名，目的是‘打破.com域名垄断的局面‘，很多人因此认为顶级域名战国时代来临

# 分类

  • 域名的第一级是顶级域，目前顶级域名有上百个，但是常用的不足10个，包括：

    • 通用：.com,.net,.org
    • 国家和地区：.cn，.us
