# svg (可缩放矢量图像)

其他图片格式是告诉计算机在屏幕上哪一点应该绘制什么颜色，
而svg是告诉计算机如何通过它的组成部分重组图形。

svg的渲染模型被称为画家模型：上层会覆盖下层

z-index paint-order 允许你改变渲染规则，由svg2引入
